## スコープ

yas80 は次のスコープがあり、
異なるスコープ間で同じ名前（識別子）を使用することが可能です。

内容はそれぞれの疑似命令を参照ください。

| 疑似命令     | スコープ     | 説明 | 専用識別子 | ネスト可否 |
| --           | --           | --   | - | :-: |
| PROC         | PROC 定義内                    | あり ( . )| 不可 |
| マクロ呼出し | MACRO で定義したマクロの呼出し | あり (@)  | 可   |
| REPT 呼出し  | REPT の各展開                  | あり (@)  | 可   |
| FUNC 呼出し  | FUNC で定義した関数の呼び出し  | なし      | 可   |
| ENUM 定義    | ENUM 要素                      | なし（※）| 可   |

※ENUM 要素定義時に他の要素を参照する場合を除く
